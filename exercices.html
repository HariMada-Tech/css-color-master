Niveau Débutant

Objectif : maîtriser les bases : color, background-color, nom de couleur et HEX simple.

1. Quelle propriété change la couleur du texte ?

Correction
p { color: blue; }

2. Quelle propriété change la couleur de fond ?

Correction
.box { background-color: yellow; }

3. Quel code HEX représente le rouge pur ?

Correction

#FF0000 = rouge maximum, vert 0, bleu 0.

4. Dans un code HEX, quelle est la valeur la plus faible ?

Correction

La valeur la plus faible est 00. La valeur la plus forte est FF.

Niveau Intermédiaire 🔒

Objectif : comprendre RGB, RGBA, transparence et aperçu après validation.

1. Quel format permet d’ajouter facilement une transparence ?

Correction
background-color: rgba(255, 0, 0, 0.2);

2. Tapez une couleur CSS pour obtenir du bleu

L’aperçu change seulement après validation.

Aperçu après validation
Correction

Exemples de réponses acceptées : blue, #0000ff, #00f, rgb(0,0,255).

3. Dans rgba(0,0,0,0.5), quelle valeur indique la transparence ?

Correction

La valeur 0.5 indique l’opacité/transparence.

4. Quel format utilise Rouge, Vert et Bleu ?

Correction

RGB signifie Red, Green, Blue : rouge, vert, bleu.

Niveau Avancé 🔒

Objectif : utiliser HSL, variantes, combinaison professionnelle et choix design.

1. Quel format est pratique pour créer une couleur plus claire ou plus foncée ?

Correction

HSL permet de modifier la luminosité avec la troisième valeur.

2. Dans un bouton, quelle propriété colore le texte ?

Correction
.btn { color: white; }

3. Quelle propriété permet d’ajouter une ombre colorée ?

Correction
box-shadow: 0 10px 25px rgba(37,99,235,.25);

4. Dans hsl(220, 90%, 60%), combien de valeurs principales trouve-t-on ?

Correction

HSL utilise 3 valeurs : teinte, saturation et luminosité.
