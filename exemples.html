Exemples visuels d’utilisation des couleurs CSS

Cette page montre le résultat réel du code CSS. Les cartes sont maintenant adaptées aux petits écrans sans barre horizontale.

Texte rouge

On utilise color pour changer la couleur du texte.

h2 {
  color: #dc2626;
}

Fond doux

On utilise background-color pour changer le fond.

.box {
  background-color: #fef3c7;
}

Bordure verte

La couleur peut être utilisée dans une bordure.

.box {
  border: 4px solid #16a34a;
}

Ombre colorée

RGBA est parfait pour les ombres transparentes.

.box {
  box-shadow: 0 18px 45px rgba(37,99,235,.35);
}

Dégradé

Un dégradé mélange plusieurs couleurs dans le fond.

.hero {
  background: linear-gradient(135deg,#2563eb,#9333ea);
}

Contraste

Sur un fond sombre, utilisez un texte clair.

.section {
  background:#111827;
  color:#ffffff;
}

Alerte douce

RGBA permet de créer un message visible mais pas agressif.

.alerte {
  background: rgba(249,115,22,.12);
  border: 2px solid rgba(249,115,22,.45);
}

Variante HSL

HSL aide à créer des variantes claires et sombres de la même couleur.

.titre {
  color: hsl(221,83%,53%);
}
