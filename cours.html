Comprendre les couleurs en CSS de A à Z

En CSS, la couleur sert à améliorer la lisibilité, l’identité visuelle et la beauté d’un site. On peut appliquer une couleur sur un texte, un fond, une bordure, une ombre ou même un dégradé.

1. Appliquer une couleur sur un texte

Pour changer la couleur d’un texte, on utilise la propriété color. 

h1 {
  color: blue;
}

Bonjour, ce titre est bleu

Ici, le texte du titre utilise color: blue;.

2. Appliquer une couleur de fond

Pour changer le fond d’un élément, on utilise background-color.

.bloc {
  background-color: orange;
}
Ce bloc utilise background-color: orange;

3. Appliquer une couleur sur une bordure

La couleur peut aussi être utilisée dans une bordure avec border.

.carte {
  border: 3px solid green;
}
Cette carte utilise une bordure verte.

4. Les noms de couleurs CSS

CSS accepte des noms simples comme red, blue, green, orange, black, white. C’est facile pour débuter, mais ce n’est pas très précis pour un design professionnel.

redbluegreenorangeblack
p {
  color: red;
}

5. Le code HEX

Le format HEX commence par #. Exemple : #ff0000 signifie rouge. Il est très utilisé dans les sites web, les logos et les maquettes.

.bouton {
  background-color: #2563eb;
  color: #ffffff;
}
Bouton avec fond HEX #2563eb

5.1. Comprendre les valeurs HEX : #RRGGBB

Un code HEX classique possède 6 caractères après le signe #. Ces 6 caractères sont divisés en 3 groupes de 2 :

RRRouge#FF0000
GGVert#00FF00
BBBleu#0000FF

Chaque groupe utilise les caractères 0 à 9 et A à F. La valeur la plus faible est 00 et la valeur la plus forte est FF.

/* Structure */
#RRGGBB

5.2. HEX court, HEX long et HEX avec transparence

En CSS, vous pouvez utiliser plusieurs syntaxes HEX selon votre besoin. Le plus utilisé est le format #RRGGBB, mais il existe aussi un format court et un format avec transparence.

Syntaxe Nom Exemple Utilisation
#RGB HEX court #f00 équivaut à #ff0000
#RRGGBB HEX long #2563eb format professionnel le plus courant
#RGBA HEX court avec transparence #0008 noir avec transparence
#RRGGBBAA HEX long avec transparence #2563ebcc bleu avec opacité

AA représente l’alpha, c’est-à-dire la transparence. 00 signifie complètement transparent, FF signifie totalement visible.

Conseil : pour les débutants, commencez avec #RRGGBB. Pour la transparence, utilisez plutôt rgba(), car c’est plus facile à lire.

Résumé des syntaxes de couleurs CSS

Très utilisé dans le web design et les maquettes.

Bon pour comprendre rouge, vert et bleu.

Idéal pour ajouter de la transparence.

HSL

Très pratique pour créer des variantes d’une même couleur.

6. Le format RGB

RGB signifie Red, Green, Blue. Chaque valeur va de 0 à 255. Exemple : rgb(255, 0, 0) donne le rouge. 

.alerte {
  color: rgb(255, 0, 0);
}
Texte en rouge avec RGB

Pourquoi utiliser RGB ? Quels avantages ?

RGB est utile quand vous voulez contrôler la couleur avec trois valeurs : rouge, vert et bleu. C’est aussi très pratique avec JavaScript, parce qu’un programme peut modifier facilement les nombres.

Exemple : si vous augmentez la valeur du rouge, la couleur devient plus rouge. Si vous augmentez le bleu, elle devient plus bleue. C’est donc un format logique pour comprendre la fabrication d’une couleur sur écran. 
.carte {
  background-color: rgb(37, 99, 235);
}

Quand l’utiliser ? Utilisez RGB pour les exercices, les calculs de couleur, les animations JavaScript, ou quand vous voulez expliquer clairement les valeurs rouge-vert-bleu aux étudiants.

7. Le format RGBA

RGBA ajoute une transparence. La dernière valeur va de 0 à 1. Exemple : rgba(255, 0, 0, 0.3) donne un rouge transparent.

.fond-transparent {
  background-color: rgba(255, 0, 0, 0.25);
}
Fond rouge transparent avec RGBA

Pourquoi utiliser RGBA ? Quels avantages ?

RGBA est comme RGB, mais avec une valeur supplémentaire appelée alpha. Cette valeur contrôle la transparence.

  • rgba(0, 0, 0, 1) : couleur totalement visible.
  • rgba(0, 0, 0, 0.5) : couleur à moitié transparente.
  • rgba(0, 0, 0, 0) : couleur invisible.

Cas très utile : créer une ombre douce, un fond transparent sur une image, une alerte légère ou un effet de survol.

.message {
  background-color: rgba(37, 99, 235, 0.12);
  border: 1px solid rgba(37, 99, 235, 0.35);
}
Exemple : fond bleu léger avec bordure transparente.

8. Le format HSL

HSL signifie Hue, Saturation, Lightness : teinte, saturation et luminosité. C’est très pratique pour créer plusieurs variantes d’une même couleur.

.titre {
  color: hsl(220, 80%, 50%);
}

Titre avec HSL

Pourquoi utiliser HSL ? Quels avantages ?

HSL est très apprécié en design parce qu’il permet de modifier une couleur de manière naturelle.

  • Hue : la teinte, par exemple bleu, rouge ou vert.
  • Saturation : couleur vive ou couleur grise.
  • Lightness : couleur claire ou sombre.

Avec HSL, vous pouvez garder la même couleur principale et créer facilement une version claire pour le fond, une version normale pour le bouton et une version sombre pour le texte.

clairmoyen clairprincipalsombretrès sombre

9. Comparaison rapide : Nom, HEX, RGB, RGBA, HSL

Nom de couleur

Avantage : très facile pour débuter.

Limite : peu précis pour un design professionnel.

color: blue;

HEX

Avantage : court, propre et très utilisé dans les maquettes, logos et sites web.

Idéal pour : couleurs fixes d'une charte graphique.

color: #2563eb;

RGB

Avantage : facile à manipuler avec JavaScript parce que les valeurs sont des nombres.

Idéal pour : générateur de couleurs, animations et exercices.

color: rgb(37, 99, 235);

RGBA

Avantage : ajoute la transparence.

Idéal pour : ombres, fonds transparents, hover, overlay sur image.

background: rgba(37,99,235,.15);

HSL

Avantage : excellent pour créer une famille de couleurs claire, normale et sombre.

Idéal pour : design system et palettes professionnelles.

color: hsl(221, 83%, 53%);

10. Exemple réel : bouton professionnel

Un bouton professionnel utilise souvent une couleur principale, un texte bien contrasté, une ombre légère et un effet au survol.

.btn {
  background-color: #2563eb;
  color: white;
  box-shadow: 0 10px 25px rgba(37, 99, 235, 0.25);
}

.btn:hover {
  background-color: #1d4ed8;
}
Exemple de bouton
